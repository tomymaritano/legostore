# Lego Store

Lego Store is a simple e-commerce application built with React and Chakra UI. It lets you browse Lego sets, add them to a cart and review your order.

## Prerequisites

- [Node.js](https://nodejs.org/) 14 or higher
- npm (comes with Node.js)

## Setup

Clone the repository and install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

The site will be available at [http://localhost:3000](http://localhost:3000).

## Running tests

Execute the unit tests with:

```bash
npm test
```

## Building for production

Generate an optimized build in the `build` directory:

```bash
npm run build
```

This command bundles the React app in production mode so it can be deployed.
